# 永磁同步电机初始位置定位，给-90度的电角度实现校准d轴，使产生力矩最大

## 概念解析

对[永磁同步电机](https://zhida.zhihu.com/search?q=永磁同步电机&zhida_source=entity&is_preview=1)进行控制的时候，一般将转子磁极方向和alpha轴（亦即A相轴）正向重合的位置作为转子零位（转子在此位置时角度为0°），如图1所示。然后[控制算法](https://zhida.zhihu.com/search?q=控制算法&zhida_source=entity&is_preview=1)依据FOC产生一个超前90°的电流is来拖动电机转起来，但是上电时候编码器值为0，无法确定真实的电角度，因此需要对d轴校准。产生一个0°电角度的信号，d轴与α轴对齐，这样就完成了电角度的对齐。并且记录该数值，作为一个差值补偿进入以后FOC计算中，因为当产生对齐电角度的差值为恒定，但是不对电机进行使能后，d轴偏差又会存在。

void DRV_MotorAlign(DRV_MotorTypeDef*v )
{
	//判断当前状态，小于最大次数跳到下一步
	if (v->alignTicker < v->alignTime)
	{
		v->alignTicker++;
		v->stateNext = Align_Mode;
			

	}
	else if ( v->alignTicker>= v->alignTime)
	{
		v->stateNext = Current_Mode;
	}

//缓慢提高q轴电流,以斜率增加
	if(v->alignId < v->alignIdref)
	{
		v->alignId += (0.0006f);
	}
//大于限幅
	else if(v->alignId >= v->alignIdref)
	{
		v->alignId = v->alignIdref;
	}
//刚开始小于一半总时间，电角度锁在90度
if(v->alignTicker <= (v->alignTime>>1))	
{
	v->alignElecTheta = (0.25f);
}
else if (v->alignTicker > (v->alignTime>>1) && v->alignTicker < v->alignTime)
{
	v->alignElecTheta -= (0.00005f);
	if (v->alignElecTheta <= 0)
	{
		v->alignElecTheta=0;
	}
}

//q轴给0
	v->alignIq = 0;
//电角度偏差值,锁紧状态可能为3999即偏差为负值，反之直接取
	motorParameter.CalibratedDegree = (TIM2->CNT >motorParameter.LineEncoder *2 )?\
	(TIM2->CNT - motorParameter.LineEncoder *4) : TIM2->CNT;
	
}

q轴始终为0，d轴与电角度缓慢变化，读取编码器CNT计数值作为差值，判断此差值为顺着电机方向逆时针转为正，逆着电机方向顺时针为负值（编码器计数方向规定），则可以取出偏差值

### 1. **定义目标**

确定对齐过程的目标，包括：

- **最终d轴电流值（Id_ref）**：期望在对齐结束时达到的电流值。
- **对齐时间（T_align）**：希望完成对齐过程的时间。

### 2. **电机模型参数**

从电机模型中获取必要的参数：

- **电机的额定电流**。
- **电机的时间常数**（如电感L和电阻R，用于计算电流的动态响应）。

### 3. **计算斜率**

使用以下公式计算电流增加的斜率：

![$\text{斜率} = \frac{\text{Id\_ref} - \text{Id\_initial}}{T\_align}$](http://chatai.qizaituan.com/img/latex-placeholder.png)斜率=*T*_*a**l**i**g**n*Id_ref−Id_initial

其中：

- **Id_initial** 是开始时的d轴电流（通常为0）。
- **Id_ref** 是目标d轴电流。
- **T_align** 是对齐时间。

### 4. **考虑电机动态响应**

根据电机的动态特性，可能需要调整斜率。电机的响应由其时间常数决定，考虑以下公式：

![$I(t) = I_{\text{ref}} \left(1 - e^{-\frac{t}{\tau}}\right)$](http://chatai.qizaituan.com/img/latex-placeholder.png)*I*(*t*)=*I*ref(1−*e*−*τ**t*)

其中：

- **I(t)** 是时间t时的电流。
- **I_ref** 是目标电流。
- **τ** 是电机的时间常数，通常由电感L和电阻R决定，![$\tau = \frac{L}{R}$](http://chatai.qizaituan.com/img/latex-placeholder.png)*τ*=*R**L*。

调整斜率和总值以确保电流在对齐过程中平稳增长，并在预定时间内达到目标值，符合电机的动态响应特性。

最终d轴电流值（Id_ref）：期望在对齐结束时达到的电流值。 对齐时间（T_align）：希望完成对齐过程的时间。这两个时间我要怎么确定

确定最终d轴电流值（Id_ref）和对齐时间（T_align）涉及以下步骤：

### 1. **确定最终d轴电流值（Id_ref）**

- **电机特性**：参考电机的额定电流和磁场特性。对于一些电机，d轴电流值通常需要设置为零或接近零，以避免磁饱和或不必要的磁场干扰。
- **应用要求**：根据应用需求，决定所需的d轴电流值。例如，在某些应用中，d轴电流可能需要在启动时达到特定值以保证稳定运行。
- **厂商建议**：查阅电机厂商的技术手册，获取推荐的d轴电流值或对齐要求。

### 2. **确定对齐时间（T_align）**

- **电机动态特性**：根据电机的时间常数（由电感L和电阻R决定）来估算对齐时间。时间常数 ![$\tau = \frac{L}{R}$](http://chatai.qizaituan.com/img/latex-placeholder.png)*τ*=*R**L* 影响电流达到稳态的速度。
- **系统响应**：选择对齐时间应考虑系统的响应时间和稳定性需求。对齐时间过短可能导致系统不稳定，对齐时间过长会影响启动速度。
- **实验调整**：进行实验测试，通过调整对齐时间来优化电机的性能。可以从电机厂商推荐值开始，然后根据实际运行情况进行调整。

假设你的电机时间常数 ![$\tau$](http://chatai.qizaituan.com/img/latex-placeholder.png)*τ* 为0.1秒：

- 如果希望电流在对齐时间内（例如2秒）达到目标值，可以选择一个适当的d轴电流（Id_ref）和对齐时间（T_align）。
- 计算公式示例：

![$  T_{\text{align}} \approx 5 \times \tau  $](http://chatai.qizaituan.com/img/latex-placeholder.png)*T*align≈5×*τ*
